Documentation Intégration (API & MCP)

MD to PDF Pro est conçu pour être facilement intégré via son API RESTful ou via le Model Context Protocol pour vos agents IA.

1. Génération de PDF via JSON

POST /api/v1/convert

Format de la requête :

{
  "markdown": "# Titre\n\nVoici le texte à convertir en PDF."
}

Exemple cURL :

curl -X POST http://localhost:5001/api/v1/convert \
-H "Content-Type: application/json" \
-d '{"markdown": "# Bonjour"}'

2. Génération de PDF via Fichier (Upload)

POST /api/v1/convert

Envoyez un fichier .md via le champ file (multipart/form-data).

Exemple cURL :

curl -X POST http://localhost:5001/api/v1/convert \
-F "file=@mon_document.md"

🤖 Intégration Agent IA (MCP)

Vous pouvez connecter cet outil directement à Claude Desktop ou tout autre agent supportant le MCP.

Ajoutez ceci à votre configuration :

"mcpServers": {
  "inspirify-pdf": {
    "command": "python",
    "args": ["/chemin/absolu/vers/backend/mcp_server.py"]
  }
}
